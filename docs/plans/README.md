## Deferred Features

The following are intentionally out of scope until needed:

- OAuth / Social Login (`omniauth-google-oauth2`)
- Role-based Authorization (`pundit` or custom)
- Background Jobs (`sidekiq`)
- Pagination (`kaminari` or `pagy`)

**Note**: Devise is not used in this project.
